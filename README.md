## ✨ About

**Red Theme** is a custom dark color theme for Visual Studio Code and compatible editors (VSCodium, Gitpod, Eclipse Theia, etc.), built with rich red tones to create a bold and focused coding environment.

Perfect for developers who want their editor to stand out with a strong, vibrant aesthetic.

---

## 📦 Installation

### From Open VSX Registry

1. Open your editor (**VS Code**, **VSCodium**, etc.)
2. Go to **Extensions** (`Ctrl+Shift+X`)
3. Search for **`Red Theme`** by `aysesudeozden`
4. Click **Install**

Or install directly via the [Open VSX Registry page](https://open-vsx.org/extension/aysesudeozden/redtheme).

### Via Command Line (ovsx)

```bash
ovsx get aysesudeozden.redtheme
```

### Manual Installation (`.vsix`)

1. Download the latest `redtheme-0.0.1.vsix` file from [Releases](https://github.com/aysesudeozden/redtheme/releases)
2. Open your editor and go to **Extensions** (`Ctrl+Shift+X`)
3. Click the **`···`** menu → **Install from VSIX…**
4. Select the downloaded `.vsix` file and reload

---

## 🎨 Applying the Theme

1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS)
2. Type **`Color Theme`** → select **Preferences: Color Theme**
3. Choose **Red Theme** from the list
