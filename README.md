## ✨ About

**Red Theme** is a custom dark color theme for Visual Studio Code, built with rich red tones to create a bold, focused coding environment. Perfect for developers who want their editor to stand out with a strong, vibrant aesthetic.

---

## 📦 Installation

### From VS Code Marketplace *(coming soon)*

1. Open **VS Code**
2. Go to **Extensions** (`Ctrl+Shift+X`)
3. Search for **`Red Theme`**
4. Click **Install**

### Manual Installation (`.vsix`)

1. Download the latest `redtheme-0.0.1.vsix` file from [Releases](https://github.com/aysesudeozden/redtheme/releases)
2. Open VS Code and go to **Extensions** (`Ctrl+Shift+X`)
3. Click the **`···`** menu (top-right) → **Install from VSIX…**
4. Select the downloaded `.vsix` file
5. Reload VS Code

---

## 🎨 Applying the Theme

1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on macOS)
2. Type **`Color Theme`** and select **Preferences: Color Theme**
3. Choose **Red Theme** from the list
